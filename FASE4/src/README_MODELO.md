# 🧠 CardioIA – Modelo Treinado

## Informações do Modelo

O projeto CardioIA utiliza um modelo de Deep Learning baseado em Redes Neurais Convolucionais (CNN) desenvolvido para classificação de imagens médicas simuladas no contexto da disciplina de Visão Computacional.

# ⚠️ Observação Importante

O arquivo do modelo treinado não foi incluído neste repositório.

Motivo:

O GitHub possui limite de 100 MB por arquivo em repositórios convencionais.

O modelo gerado durante o treinamento possui aproximadamente 128 MB, ultrapassando esse limite.

Por esse motivo, apenas o notebook de treinamento foi disponibilizado.

---

# 🔄 Como Reproduzir o Modelo

1. Abrir o notebook:

```text
CardioIA_Fase4_Preprocessamento_FINAL.ipynb
```

2. Executar todas as células.

3. Realizar novamente:

* carregamento do dataset;
* pré-processamento;
* treinamento da CNN;
* avaliação do modelo;
* geração do arquivo .h5.
